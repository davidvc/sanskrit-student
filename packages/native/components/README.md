# Components

Reusable React Native components organized by domain.

## Structure

- `ui/` - Generic, reusable UI components (buttons, inputs, cards, etc.)
- `sanskrit/` - Sanskrit-specific components (devanagari text, transliteration displays)
- `translation/` - Translation feature components (translation cards, word lookup)
- `ocr/` - OCR and camera-related components (camera views, text recognition displays)
