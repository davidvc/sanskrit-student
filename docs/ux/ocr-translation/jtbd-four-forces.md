## Design Implications

### Reduce ANXIETY (Critical)
1. **Show confidence scores prominently** - Build trust through transparency
2. **Visual comparison side-by-side** - Let users verify extraction even without reading Devanagari
3. **Warn before problems occur** - "Tip: Use bright lighting" upfront
4. **Actionable error messages** - Specific guidance, not generic failures

### Overcome HABIT (Important)
1. **Make OCR dramatically faster than workarounds** - 10x speed advantage
2. **Demonstrate accuracy immediately** - First experience must succeed
3. **Integrate with existing study workflow** - Copy/paste to notes (don't force new habits)

### Amplify PULL (Essential)
1. **Instant gratification** - Photo â†’ translation in under 5 seconds
2. **Empowerment messaging** - "Access any Sanskrit text" not "OCR tool"
3. **Respect sacred context** - Design language reflects reverence for texts
